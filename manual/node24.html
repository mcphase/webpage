Example - a simple antiferromagnet

In the following description of the input files we will always refer to a simple example: a simple antiferromagnet on a primitive orthorhombic lattice. The first time user will thus have a simple example to follow, all corresponding files are given in the directory tutorial/03magnetic_phases_mcphas/simpleAF.
