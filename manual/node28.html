Single Ion Property Input Files

In order to speed up calculations or treat special problems a large variety of single ion modules is available. This includes the option to load a user written single ion module. Details are given in chapter 13.

The first time user of McPhase should use the module so1ion and create an appropriate single ion property input file as described in section 6.3. A good starting point are several examples given in directory examples.
