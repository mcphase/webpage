Workshop Aims

This workshop aims to give a platform for the discussion of special topics in using the Mcphase Software Package. For newcomers to McPhase there will be an introduction to some basic concepts.
The small number of participants allows to focus on individual research topics and discuss basics and useful simulation techniques.
